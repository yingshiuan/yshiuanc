  • Robotic Fabrication
    of
    Brick Wall


    On-site robotic fabrication via locating system of HTC VIVE Tracker


  • The aim of this project is to fabricate a fundraising brick wall measuring 3 meters in height and 6.7 meters in length using an HTC VIVE tracker which locates relative coordinates between the double-curved brick wall and the UR10 robot on site.


  • Fundraising Brick Wall

  • In this project, my role is to fabricate the fundraising brick wall on site in two month and develop robotic arm (UR10) and locating system of HTC VIVE Tracker.

    This project started with two questions: How to move a robotic system (UR10) on a construction site and how to use it to position building components in-situ.

  • On-site Robotic fabrication via locating system of HTC VIVE Tracker

  • We mounted our Robotic arm U10 on a mobile station to make it accessible on site. A pick-up location for a 20x10x5cm wooden brick is set up at the bottom of the mobile station.

  • Robotic fabrication setup

  • We collect tracking data with an HTC VIVE Tracker locating system. Using a technology stack comprising of the modelling software Rhino3D, its developed platform Grasshopper, SteamVR, OpenVR and Python, we create a virtual model of the environment.

  • Control Framework & Communication Protocol

  • The VIVE Tracker extracts the relative coordinates of the UR10 and the reference brick. We then use this data to simulate the following fabrication process: Place 80 to 120 bricks over a 1 to 2 hour time period without the UR10 colliding while being moved to a new location on site.

  • Real-time Simulation

  • Under the constraint of the ceiling height, we moved the robotic arm 22 times to build the 24 layers of wall comprising of 1024 bricks. The rest of 28 layers with 406 bricks were built by prefabricated elements.

  • Sequence

  • This approach provides digital integration and informational oversight across the entire design and execution phases of the robotic fabrication process
